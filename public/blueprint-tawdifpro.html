تحسين SEO & الأداء

مستمر
  • ✅ Metadata الديناميكي (title, description لكل صفحة)
  • ✅ Open Graph + Twitter Cards
  • ✅ Schema.org Organization (موجود ف layout)
  • ✅ FAQ Schema (موجود ف المقالات + حاسبة الراتب)
  • ✅ Sitemap.xml
  • ✅ Robots.txt
  • ❌ Article Schema (nécessaire pour chaque article)
  • ❌ BreadcrumbList Schema
  • ❌ Google Analytics / Search Console
  • ❌ تحسين Core Web Vitals (LCP, CLS, INP)
  • ❌ Images optimisées (next/image, WebP)
  • ✅ Cairo font swap (pas de CLS)
  • ❌ إضافة canonical URLs صريحة
  • ❌ Internal linking strategy (ربط المقالات ببعضها)
